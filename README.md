# Sitio Web Institucional — Unidad Educativa Municipal Borja

Este proyecto corresponde al desarrollo del sitio web institucional de la **Unidad Educativa Municipal Borja**, realizado como parte del proyecto de vinculación con la comunidad de la Universidad UEES.

El objetivo principal fue crear una plataforma digital clara, funcional y accesible que permita a la institución fortalecer su presencia en línea y mejorar la comunicación con estudiantes, docentes y padres de familia.

## 🎯 Objetivos del Proyecto

- Diseñar y desarrollar un sitio web institucional moderno y fácil de navegar.  
- Organizar y presentar la información oficial proporcionada por la institución.  
- Facilitar el acceso a datos relevantes como historia, misión, visión, oferta académica y proceso de inscripción.  
- Fortalecer la presencia digital de la Unidad Educativa Municipal Borja.
